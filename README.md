# Order Control Tower Pilot

Interactive React/Vite mock-up for a live O2P exception-management Control Tower.

## Features

- Management KPI overview
- Live filter mock-up for Region, MO, Priority and Promise Date
- Prioritized reference worklist
- Cross-order impact column for shared dependencies
- Grouped exception profile
- Clickable operational action pages
- Dedicated put-away scenario showing that missing inbound stock for Article 234 – Drill Set affects five additional express orders due today
- Responsive layout for desktop and tablet

## Local start

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## GitHub and Vercel

1. Create a new GitHub repository.
2. Upload all files and folders from this project root.
3. Import the repository into Vercel.
4. Framework preset: Vite.
5. Build command: `npm run build`.
6. Output directory: `dist`.
