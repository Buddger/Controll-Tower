# Order Control Tower

Single-file React/Vite pilot application. The complete dashboard and action-page logic is contained in `src/App.jsx`.

## Local start

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## GitHub and Vercel

Upload the contents of this folder directly to the root of the GitHub repository. In Vercel select the Vite framework preset. The expected output directory is `dist`.
